WAYA

Forgiving the name of a friend's wife or child can be really embarrassing. WAYA (Who Are You, Again) is a software to help you avoid this situation. With WAYA you can take notes on your friends, colleagues and people you hold dear, to avoid having to rely on your (bad) memory.

Graphical interface

Content of the interface

  • Left: List of the persons
  • Left-bottom: category filter, to only display persons from a specific category in the list
  • Center: main panel. Depending on the cases it displays:
    • Home : when no person is selected
    • Person's profile: when a person has been selected
    • Help (current window): when the help has been opened

Home

The home contains shortcuts for core features

  • Creation of a person
  • Showing the help
  • Showing information about the software

Profile of a person

The first part of the profile contains

  • the name of the person (can be edited freely)
  • the photo, which can be modified using the "Select" button below
  • a "Tags" button which allows to set tags (existing or new) for this person
  • a "Close" button, to close the profile and come back to the home
  • a "Delete" button, to delete once and for all the person

The second part of the profile contains information about the person. Each piece of information has a name (on the left), and one or several entries. All the fields can be edited freely.

The "Add information" button on the bottom of the profile is used to create a new piece of information for this person.

Buttons around a piece of information:

  • X : removal of the piece of information and its entries
  • - : removal of the entry on the left of the button
  • + : addition of an entry to the piece of information
